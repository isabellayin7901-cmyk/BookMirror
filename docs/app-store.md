# 上架 App Store 清单

## 一、账号 & 法律

- [ ] **Apple Developer Program**：$99/年（个人或公司）
- [ ] **D-U-N-S 号**（仅公司账号需要）
- [ ] **隐私政策 URL**（公网可访问）—— 用 termly.io / iubenda 生成
- [ ] **用户协议 URL**
- [ ] **App Privacy 数据收集声明**：MBTI 答案、目标、问题、自由输入文本都属敏感数据，必须勾选并说明用途
- [ ] **年龄分级**：建议 17+ 避免 COPPA 麻烦

## 二、AI 合规

- [ ] 遵守 [Anthropic Usage Policy](https://www.anthropic.com/legal/aup)
- [ ] **禁止**：把推荐包装成医疗/心理治疗建议
- [ ] 文案明示："推荐由 AI 生成，仅供参考，不构成专业心理建议"
- [ ] 涉及"情绪""内耗""焦虑"等词时，结果页底部加链接：「严重情绪问题请寻求专业帮助」

## 三、技术准备

- [ ] **API key 不能放客户端**：所有 Claude 调用走自己的后端代理
- [ ] **限流**：每用户每天 N 次免费推荐，防滥用
- [ ] **EAS Build**：Expo 云打包出 .ipa
- [ ] **Bundle ID**：`com.yourname.bookmirror`
- [ ] **图标**：1024×1024 PNG
- [ ] **启动屏**
- [ ] **截图**：6.7"、6.5"、5.5" 三套（iPhone Pro Max / Plus / 8 Plus）
- [ ] **TestFlight 内测**：5–10 人，跑通至少 3 天

## 四、登录 & 支付

- [ ] 如有第三方登录（Google / 微信），**必须**同时支持 Sign in with Apple（硬性要求）
- [ ] 订阅必须用 **苹果 IAP**，不能跳支付宝/微信，否则被拒
- [ ] 苹果抽成 30%（小开发者计划首年 15%）

## 五、容易被拒的点

| 雷区 | 规避 |
|---|---|
| AI 推荐写成"算命""测命运" | 改成"了解阅读偏好""个性化推荐" |
| MBTI 测试包装成心理诊断 | 加免责声明 |
| 没有 Sign in with Apple | 加上 |
| 隐私政策链接 404 | 上线前测试 |
| 应用太空（功能不够） | 至少 4 页面 + 真实内容 |

平均 2–3 轮过审，每轮 1–3 天。

## 六、国内上架（建议第二版再做）

- [ ] 软件著作权（淘宝代办 ~¥500，30 天）
- [ ] ICP 备案
- [ ] 国内 App Store 单独提交

**建议**：第一版只上海外区（美国 / 香港 / 新加坡），用英文优先 + 中文可选，验证产品再考虑国内。

## 七、提交后

- [ ] 准备 App Store 描述（中英）
- [ ] 关键词：`reading recommendation, MBTI, personality, book, AI, self-growth`
- [ ] 客服邮箱
- [ ] 营销 URL（落地页，可选）
